### 1- Project Overview

This project showcases an end-to-end Power BI dashboard built to monitor and analyze retail business performance. The dashboard provides insights into key metrics such as total transactions, revenue, profit, and return rates across time and geography.

Advanced DAX measures were implemented to support time-based analysis, including month-to-date (MTD), rolling 60-day revenue, and previous month comparisons. KPI visuals were designed to evaluate performance against targets, with conditional formatting applied to reflect business logic (e.g., lower return rates are considered favorable).

The report also includes a geographic map for store-level analysis and a weekly revenue trend chart to highlight seasonality and performance fluctuations throughout the year.

### 2- Key Features:

- Interactive KPI cards with variance vs target
- Time intelligence:
    - Month-to-date (MTD)
    - Previous month comparison
    - Rolling 60-day revenue
- Conditional formatting with business-driven logic
- Geospatial analysis using map visual (transactions by city)
- Weekly revenue trend analysis
- Dynamic filtering with slicers (country, hierarchy drill-down)

### **3- Tools & Skills:**

- Power BI
- DAX (CALCULATE, SUMX, PREVIOUSMONTH, DATESINPERIOD, TOTALMTD)
- Data modeling (star schema, relationships, inactive relationships)
- Data visualization & dashboard design

### **4- Key Learning:**

- Strengthened understanding of filter context and time intelligence in DAX
- Learned how to model business logic into visuals (e.g., handling metrics where lower values indicate better performance)
- Improved dashboard design skills for clarity, usability, and storytelling
